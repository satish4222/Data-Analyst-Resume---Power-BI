# Power BI Resume Dashboard

## Project Overview

The **Power BI Resume Dashboard** is a data-driven portfolio project that uses **Power BI** to present personal, educational, and professional information in a clear and interactive format. The dashboard connects to a structured **MySQL-based resume database** and visualizes details such as skills, certifications, education, projects, and publications.

This project demonstrates the ability to design databases, connect them to Power BI, and build meaningful dashboards suitable for portfolio and interview presentation.

---

## Project Goal

* To transform resume data into an **interactive Power BI dashboard**
* To demonstrate **end-to-end BI skills**, from database design to visualization
* To present professional information in a **clear, analytical, and visual manner**
* To create a **portfolio-ready project** for data analyst and BI roles

---

## Project Files

* **satish BI resume.sql** – SQL script to create and populate the MySQL database
* **satish resume.pbix** – Power BI dashboard file
* **README.md** – Project documentation

---

## Database Schema

The MySQL database consists of the following tables:

* **satish** – Personal information such as name, phone number, and email
* **skills** – Skill names with proficiency levels (rated from 1 to 5)
* **certifications** – Certifications earned, associated skills, and completion dates
* **education** – Academic records including school and college scores or percentages
* **projects** – Academic and personal projects with descriptions and completion dates
* **publications** – Published research papers with titles and publication dates

---

## Insights from the Dashboard

The Power BI dashboard provides the following insights:

* **Personal Overview** – Displays name, contact number, and email address
* **Skill Matrix** – Visual representation of skill proficiency levels
* **Certifications Timeline** – Timeline view of certifications earned over time
* **Education Summary** – Comparison of academic performance across institutions
* **Project Showcase** – Visual summary of completed academic and personal projects
* **Publications** – Overview of published research papers with timelines

## Tools and Technologies Used

---

## How to Use the Project

1. Run the SQL script **satish BI resume.sql** in MySQL to create and populate the database
2. Open **satish resume.pbix** in Power BI Desktop
3. Connect Power BI to MySQL using **Get Data → MySQL Database**
4. Refresh the data to load live information
5. Explore the dashboard using filters and interactive visuals
